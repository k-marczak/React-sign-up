# Sign-up application

---

## Description


Sign-up application written in React using MongoDb database. 

---

## Technologies

- React
- Express.js
- Bootstrap
- MongoDB
- Axios

---

## Setup

To test this app install dependencies

In .env folder include username, password and dbname.

To run this project

```
$ npm install
$ npm start
```
